
## void — Minimalist Social Platform
A clean, focused, and text-centric social media dashboard prototype. void is designed for high-signal interaction, stripping away the noise of modern platforms to provide a "breathing room" for thoughts, code snippets, and design principles.

## Core Philosophy
"Minimalism isn't the absence of things; it's the presence of the right things."

The platform is built on the idea that every pixel costs attention. It features a monochrome-inspired aesthetic, utilizing a grayscale palette that shifts seamlessly between light and dark modes to keep the focus entirely on content.

### Key Features
Multimodal Posting: Support for text-based posts (up to 280 characters) with integrated media handling for images, videos, and audio clips.

Media Interaction: Features custom expanded states for media, including grayscale image previews and native-styled audio/video players.

Stalk Mode: A unique profile interaction that allows users to "stalk" a profile, providing a persistent visual indicator while browsing that specific user's timeline.

Direct Messaging: A full-featured messaging interface with real-time conversation views and unread status indicators.

Responsive Dashboard: Includes a dynamic sidebar for navigation, a central feed, and a right-side panel for activity stats and trending topics.

Theme Engine: Built-in support for high-contrast light and dark themes using CSS variables for instant state changes.

### Technical Stack
Frontend: Pure HTML5 and CSS3 (no external UI frameworks).

Styling: CSS Custom Properties (Tokens) for theme management and flexbox-based layouts.

Interactions: Vanilla JavaScript for state management, view routing, and modal handling.

Icons: Hand-crafted inline SVGs for a lightweight, dependency-free experience.
